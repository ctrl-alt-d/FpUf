# Subdivisió de xarxes IP (versió 4)
## SMX-MP05-UF2 - Exercici de configuració de commutadors i encaminadors.
##Recursos
[IP Subnet Calculator](http://www.subnet-calculator.com/)

##Activitat

**Exercici 1.** Omple la següent taula. Suposa que cada IP té la seva corresponent màscara de subxarxa per defecte.

| Adreça IP     | És IP de host? (S/N)  | Raona-ho  |
| ------------- |:-------------:| -----:|
| 150.100.255.255| |  |
| 175.100.255.18      |      |   |
| 195.234.253.0 |     |     |
| 100.0.0.23 |     |     |
| 188.258.221.176 |     |     |
| 127.34.25.189 |     |     |
| 224.0.0.2 |     |     |

**Exercici 2.** Volem subdividir la xarxa 192.168.2.0 en 8 subxarxes diferents:

a) Quants bits de subxarxa necessitem? Quants hosts hi haurà en cada subxarxa?

b) Com quedarà la màscara de subxarxa?

c) A quina subxarxa pertany la IP 192.168.2.70? (heu de trobar l'adreça de xarxa corresponent) Quina adreça de difusió li correspon a aquesta IP?

d) Repetiu l'apartat anterior però amb la IP 192.168.2.136

**Exercici 3.** Passa a format de barra les següents màscares de subxarxa:

- 255.0.0.0
- 255.255.0.0
- 255.255.255.0
- 255.248.0.0
- 255.255.192.0
- 255.255.255.128
- 255.255.255.224

**Exercici 4.** Passa ara les següents màscares de subxarxa a la seva notació IP decimal:

- /10
- /16
- /19
- /27
- /30

**Exercici 5.** Quines de les següents adreces IP es poden usar com adreces de host en la xarxa 192.168.8.0/29? Justifica cada resposta.

a) 192.168.8.7

b) 192.168.8.8

c) 192.168.8.9

d) 192.168.8.3

e) 192.168.8.252
